# MyMileage - Every Mile Counts — Feature Guide

---

## 1. 목표 리마인더 알림 (Goal Reminder Notifications)

### 개요

매월 말 러닝 목표 미달성 시, 설정에서 활성화된 알림이 오전 9시에 자동 발송됩니다.
알림은 **D-7 / D-3 / D-1** 세 종류이며, 각각 독립적으로 켜고 끌 수 있습니다.

---

### 알림 종류별 레이아웃

#### D-7 알림 (월말 7일 전 오전 9시)

```
제목: 🏃 Monthly Goal Reminder
본문: 12.5km remaining. Run in 7 days! 💪

(한국어 예시)
제목: 🏃 이번 달 목표 리마인더
본문: 목표까지 12.5km 남았습니다. 7일 안에 달려보세요! 💪
```

#### D-3 알림 (월말 3일 전 오전 9시)

```
제목: 🏃 Monthly Goal Reminder
본문: 8.3km remaining. Run in 3 days! 💪

(한국어 예시)
제목: 🏃 이번 달 목표 리마인더
본문: 목표까지 8.3km 남았습니다. 3일 안에 달려보세요! 💪
```

#### D-1 알림 (월말 전날 오전 9시)

```
제목: 🏃 Monthly Goal Reminder
본문: 4.1km remaining. Last chance tomorrow! 💪

(한국어 예시)
제목: 🏃 이번 달 목표 리마인더
본문: 목표까지 4.1km 남았습니다. 내일이 마지막 기회예요! 💪
```

> **목표 유형별 단위 표시**
>
> - 거리 목표: `km` 또는 `mi` (설정된 거리 단위 기준)
> - 칼로리 목표: `kcal`
> - 시간 목표: `hr` / `시간` / `Std` 등 언어별 표기

---

### 트리거 조건

| 조건             | 설명                                                           |
| ---------------- | -------------------------------------------------------------- |
| 알림 전체 켜짐   | 설정 → 알림 → "목표 리마인더 알림" 토글 ON                   |
| 해당 알림 켜짐   | D-7 / D-3 / D-1 중 해당 항목 토글 ON                           |
| 목표 미달성      | 남은 달성량 > 0 (이미 목표 달성 시 알림 미발송)                |
| 발송 시점 미도래 | 예약 시점이 현재 시각보다 미래여야 함 (당일 9시 이후이면 스킵) |

**알림 재스케줄 시점**: 동기화 완료 후, 설정 저장 시 자동으로 기존 알림 취소 후 재등록.

---

### 동작 흐름

```
HealthKit 동기화 완료
  → 남은 달성량 계산 (goalTarget - currentValue)
  → 남은량 > 0 이면 알림 스케줄 실행
    → D-7: 이번달 1일 + 1달 - 7일 = 날짜 계산
    → D-3: 이번달 마지막일 - 2일
    → D-1: 이번달 마지막일
  → 각 알림: 해당 날짜 09:00 에 UNCalendarNotificationTrigger 등록
```

## 2. 스트릭 지표 설명 (Streak Metrics)

대시보드 하단의 스트릭 카드는 세 가지 지표를 보여줍니다.

```
┌──────────────────────────────────────────────┐
│  🔥 연속 달성  │  🏆 최장 기록  │  ✅ 누적 달성  │
│   3 개월       │   5 개월       │   12 개월      │
└──────────────────────────────────────────────┘
```

---

### 🔥 연속 달성 (Streak / Current Streak)

**정의**: 이번 달 포함, 연속으로 월 목표를 달성한 개월 수.

**예시**:

- 1월 달성 → 2월 달성 → 3월 달성: `연속 달성 = 3개월`
- 4월 미달성 → 5월 달성: `연속 달성 = 1개월`

**시각화**:

- 값이 1 이상일 때 🔥 아이콘과 숫자가 **강조 색상**으로 표시
- 값이 0일 때 `-` 표시 (회색)
- **연속 3개월 이상** 달성 시 스트릭 카드 외곽에 강조 색상 테두리 표시 (성취감 시각화)

**달성 판정 기준**: 해당 월의 동기화 완료 시점에, `displayCurrentValue(distanceUnit:) >= goalTarget` 이면 달성.

---

### 🏆 최장 기록 (Best / Longest Streak)

**정의**: 지금까지 기록한 가장 긴 연속 달성 개월 수.

**예시**:

- 과거 최고 기록이 5개월 연속이었고, 현재 스트릭이 3개월이라면: `최장 기록 = 5개월`
- 현재 스트릭이 최장 기록을 갱신하면 실시간으로 업데이트

**시각화**:

- 🏆 아이콘 + 숫자 (회색, 중립적 색상)
- 값이 0이면 `-` 표시

---

### ✅ 누적 달성 (Total / Total Goal Met Months)

**정의**: 스트릭 여부에 관계없이, 지금까지 월 목표를 달성한 **총 개월 수** 누적합.

**예시**:

- 1~12월 중 7개월 달성했다면: `누적 달성 = 7개월`
- 연속 여부와 무관하게 달성한 달이면 모두 카운트

**시각화**:

- ✅ 아이콘 + 숫자 (회색, 중립적 색상)
- 0부터 시작, 달성할 때마다 1씩 증가

---

### 스트릭 업데이트 시점

HealthKit 동기화 완료 시 자동으로 업데이트됩니다.

```
동기화 완료 → updateStreak(stats:) 호출
  → 이번 달 목표 달성 여부 판단
  → 달성 시:
      - lastGoalMetYearMonth 업데이트 (YYYY-MM)
      - 직전 달 달성 여부 확인 → 연속이면 currentStreak +1, 아니면 1로 리셋
      - longestStreak 갱신 (currentStreak > longestStreak 이면 업데이트)
      - totalGoalMetMonths +1
  → 미달성 시: currentStreak 리셋 없음 (월말 이전이므로 아직 기회 있음)
```

> **참고**: 같은 달에 여러 번 동기화해도 스트릭은 중복 카운트되지 않습니다 (`lastGoalMetYearMonth` 체크).

## 3. Pro 인앱 구매 (Pro In-App Purchase)

### 개요

MyMileage Pro는 일회성 $1.99 비소모성(Non-Consumable) 구매로, 두 가지 프리미엄 기능을 해제합니다.

| 기능 | 무료 | Pro |
|------|------|-----|
| 러닝 트래킹 + 대시보드 + 로그 | O | O |
| 홈 화면 위젯 (4개 디자인) | O | O |
| 테마 색상 / 다크·라이트 / 언어 | O | O |
| 목표 리마인더 알림 | O | O |
| **위젯 배경 사진 커스텀** | X | **O** |
| **잠금화면 위젯 (원형 + 직사각형)** | X | **O** |

---

### 구매 흐름

```
설정 탭 → Pro 섹션 (비구매 시에만 표시)
  → "Pro 업그레이드" 버튼 (가격 표시)
  → Apple 결제 시트
  → 성공 시:
      - isPro = true (App Group UserDefaults 저장)
      - 위젯 리로드 (잠금화면 위젯 즉시 활성화)
      - 토스트: "Pro가 해제되었습니다! 감사합니다!"
      - Pro 섹션 숨김 + 일반 섹션에 "Pro Unlocked" 배지 표시
  → 취소/실패 시: 상태 변화 없음
```

---

### 복원 흐름

```
설정 탭 → Pro 섹션 → "구매 복원" 버튼
  → AppStore.sync() 호출
  → Transaction.currentEntitlements 확인
  → Pro 구매 내역 존재 시:
      - isPro = true
      - 토스트: "구매가 복원되었습니다!"
  → 내역 없음 시:
      - 토스트: "이전 구매 내역이 없습니다."
```

---

### 기능 게이팅 상세

#### 위젯 배경 사진

- **무료 상태**: 설정 → Widget 섹션에서 "사진 선택" 항목이 흐리게 표시 + 자물쇠 아이콘 + "Pro" 뱃지
- **Pro 상태**: PhotosPicker 활성화 → 사진 선택 → 크롭 시트 → 저장 → 위젯 배경 적용
- **위젯 렌더링**: `entry.isPro && entry.bgImageData != nil` 일 때만 배경 이미지 표시

#### 잠금화면 위젯

- **무료 상태**: 위젯 갤러리에서 추가는 가능하지만, 원형 위젯은 자물쇠 아이콘만 표시, 직사각형 위젯은 "앱에서 Pro로 잠금 해제하세요" 메시지 표시
- **Pro 상태**: 원형 게이지(진행률) + 직사각형(현재값/목표 + 게이지 + 남은 일수) 정상 표시

---

### 기술 구현

| 항목 | 내용 |
|------|------|
| Product ID | `com.jangkyuju.motivationrun.pro` |
| 유형 | Non-Consumable |
| 프레임워크 | StoreKit 2 |
| 상태 저장 | `SharedDataManager.saveIsPro()` → App Group UserDefaults |
| 위젯 참조 | `SimpleEntry.isPro` 필드로 위젯에서 직접 확인 |
| 트랜잭션 리스너 | 앱 시작 시 `Transaction.updates` 자동 감지 |
| 로컬 테스트 | `Products.storekit` 파일 (Xcode Scheme 설정 필요) |

### Pro 상태와 데이터 초기화

- "전체 데이터 초기화" (Reset All Data) 실행 시 **Pro 상태는 유지**됩니다
- Pro 구매는 Apple 서버에 영구 기록되므로, 앱 삭제 후 재설치해도 "구매 복원"으로 복구 가능
